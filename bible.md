문제상황 : 640x640 3채널 크기의 이미지에 스크래치, 찍힘, 눌림 등의 불량이 존재한다. 해당 이미지에서 불량 위치를 Auto-labeling 하여야 하는데, Dino-v3를 이용하여 불량 유형 종류를 파악할 수 있어야 한다.
아래와 같은 전제가 합리적인지 파악하여야 하며, 합리적이지 않다면 그 이유를, 합리적이라면 구현 방안을 자세하게 3가지 제시하여라

대전제 : Dino v3로 픽셀에 대해서 매우 섬세하게 Class 분류가 가능하다 ( 784채널 )
중전제 : 해당 784채널 안에서 스크래치, 찍힘, 눌림 등의 불량만 필터링 할 수 있는 벡터 영역이 존재한다.
소전제 : 사람이 제시하지 않아도 1000개 이상의 패치 이미지가 주어지면 각 벡터 영역별로 유형을 분리할 수 있어야 한다.
